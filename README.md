# 🪐 Projeto Planeta

## 📖 Sobre o Projeto

O **Projeto Planeta** foi desenvolvido com o objetivo de transformar uma interface criada no **Figma** em uma página web funcional utilizando **HTML** e **CSS**.

A atividade tem como foco a interpretação de layouts, a reprodução fiel de elementos visuais e a aplicação de boas práticas de estruturação e estilização, simulando uma etapa comum do fluxo de trabalho de desenvolvedores Front-End.

---

## 🎯 Objetivos

* Interpretar layouts desenvolvidos no Figma.
* Converter protótipos em páginas web utilizando HTML e CSS.
* Trabalhar com posicionamento e alinhamento de elementos.
* Reproduzir cores, tipografia e espaçamentos do design original.
* Desenvolver atenção aos detalhes visuais.
* Praticar a construção de interfaces responsivas.
